# GameKeeper

[English](../README.md) | [简体中文](README-CN.md)

GameKeeper 是一款簡單易用的通用遊戲輔助工具，專門用於**偽造視窗前景狀態**。這使得某些原本在切換到背景時會自動暫停運行的遊戲或應用程式，能夠持續在背景中保持活躍與運行。

## 功能特色

- **直觀的操作介面**：透過簡單的視窗介面，快速找到並管理您正在執行的遊戲或應用程式。
- **即時啟用/停用**：無需重新啟動遊戲，即可隨時開啟 (Attach) 偽造前景功能，或關閉 (Detach) 恢復原狀。
- **廣泛支援**：同時支援 x86 與 x64 架構的應用程式。

## 建置說明

本專案使用 **Visual Studio 2026** 與 **.NET 8** 進行開發，請依照以下步驟進行建置：

1.  **安裝開發環境**：
    - 安裝 **Visual Studio 2026**。
    - 在 Visual Studio Installer 中勾選以下工作負載與元件：
      - **使用 C++ 的桌面開發** (Desktop development with C++)。
      - **.NET 桌面開發** (.NET desktop development)，並確認包含 **.NET 8** SDK。

2.  **執行自動建置**：
    - 雙擊專案根目錄下的 `build.bat` 檔案。
    - 等待腳本執行完畢，視窗顯示「Build and deployment complete」。
    - 建置完成的執行檔會位於 `output` 資料夾中。

## 使用說明

1.  **啟動程式**：執行 `output` 資料夾中的 `GameKeeper.exe`（建議以管理員身分執行）。
2.  **刷新列表**：按下介面上的 **Refresh** 按鈕，更新當前運行的進程列表。
3.  **啟用功能**：在選單中選取目標遊戲或應用程式，接著按下 **Attach** 按鈕。
    - _此時遊戲將被視為處於前景狀態，即使切換視窗也不會暫停。_
4.  **停用功能**：若要關閉功能，只需在已注入的列表中點擊該進程旁的 **Detach** 按鈕即可。
