# GameKeeper

[English](../README.md) | [繁體中文](README-ZH.md)

GameKeeper 是一款简单易用的通用游戏辅助工具，专门用于**伪造窗口前景状态**（Foreground Status）。这使得某些原本在切换到背景时会自动暂停运行的游戏或应用程序，能够持续在背景中保持活跃与运行。

## 功能特色

- **直观的操作界面**：透过简单的窗口界面，快速找到并管理您正在执行的游戏或应用程序。
- **即时启用/停用**：无需重新启动游戏，即可随时开启 (Attach) 伪造前景功能，或关闭 (Detach) 恢复原状。
- **广泛支持**：同时支持 x86 与 x64 架构的应用程序。

## 构建说明

本专案使用 **Visual Studio 2026** 与 **.NET 8** 进行开发，请依照以下步骤进行构建：

1.  **安装开发环境**：
    - 安装 **Visual Studio 2026**。
    - 在 Visual Studio Installer 中勾选以下工作负载与组件：
      - **使用 C++ 的桌面开发** (Desktop development with C++)。
      - **.NET 桌面开发** (.NET desktop development)，并确认包含 **.NET 8** SDK。

2.  **执行自动构建**：
    - 双击专案根目录下的 `build.bat` 档案。
    - 等待脚本执行完毕，视窗显示「Build and deployment complete」。
    - 构建完成的执行档会位于 `output` 资料夹中。

## 使用说明

1.  **启动程序**：执行 `output` 资料夹中的 `GameKeeper.exe`（建议以管理员身分执行）。
2.  **刷新列表**：按下界面上的 **Refresh** 按钮，更新当前运行的进程列表。
3.  **启用功能**：在选单中选取目标游戏或应用程序，接着按下 **Attach** 按钮。
    - _此时游戏将被视为处于前景状态，即使切换窗口也不会暂停。_
4.  **停用功能**：若要关闭功能，只需在已注入的列表中点击该进程旁的 **Detach** 按钮即可。
